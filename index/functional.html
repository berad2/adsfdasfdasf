LEGIT BOT
ПОМОЩЬ В НАВОДКЕ
  • Aimbot Active - Включить/Выключить Aim

  • Flash Check - Проверка на Flash
  • Jump Check - Проверка на прыжок

  • Bone - Кость для наводки
  • Fov - Радиус доводки

TRIGGER BOT
АВТОМАТИЧЕСКАЯ СТРЕЛЬБА
  • Active - Включить/выключить Trigger
  • On key - Настройка кнопки работы
  • Delay - Задержка перед выстрелом
  • Bone Group - Кость в какую стрелять
  • Head Only - Стрельба только в голову
VISUAL
ИНФОРМАЦИЯ О ПРОТИВНИКЕ
  • Enemy Only - Подсвечивать врагов
  • Visible Only - Только если виден
  • Box - Обводка моделек
  • Health - Показывает кол-во жизней игрока
  • Armor - Показывает кол-во брони игрока
  • Weapon - Показывает оружия игрока
  • Soud - Показывает шаги
  • Chams - Подсветка модельки игрока
  • Radar - Радар с игроками

  • Auto Pistol - Автоматическая стрельба из пистолета
